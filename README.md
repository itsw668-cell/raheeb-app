# 🍔 Raheeb — نظام الطلبات

## هيكل الملفات
```
raheeb-system/
├── app.py                  ← الخادم (Flask Backend)
├── requirements.txt        ← المكتبات المطلوبة
├── orders.db               ← قاعدة البيانات (تُنشأ تلقائياً)
└── templates/
    ├── index.html          ← واجهة الزبون (قائمة + سلة + طلب)
    ├── admin_login.html    ← صفحة دخول الأدمن
    └── admin_dashboard.html← لوحة تحكم الطلبات

```

## تثبيت وتشغيل

### 1. تثبيت Python (إذا لم يكن مثبتاً)
https://www.python.org/downloads/

### 2. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 3. تشغيل الخادم
```bash
python app.py
```

### 4. فتح المتصفح
- **واجهة الزبون:**    http://localhost:5000
- **لوحة التحكم:**     http://localhost:5000/admin

## بيانات الدخول للأدمن
- اسم المستخدم: `raheeb`
- كلمة المرور:  `raheeb2026`

⚠️ **غيّر كلمة المرور في ملف app.py قبل النشر!**

## الميزات
✅ قائمة طعام كاملة مع سلة مشتريات
✅ نموذج طلب بالبيانات الجزائرية (ولاية، بلدية، عنوان)
✅ تحقق من صحة رقم الهاتف الجزائري
✅ لوحة تحكم محمية بكلمة مرور
✅ تحديث الطلبات كل 10 ثوانٍ تلقائياً
✅ إشعار صوتي عند وصول طلب جديد
✅ تغيير حالة الطلب: جديد → تحضير → توصيل
✅ إحصائيات لحظية (جديد / تحضير / توصيل / إجمالي)
